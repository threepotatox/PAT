#### 自问自答环节：

Q：为什么会有这个库？

A：因为纯 C 解答的参考都零零散散的，所以把自己头铁写的 C 源码整理了一下

Q：为什么用 C 解答？

A：遥想当年仅和 C 处过对象，所以用 C 写了下去，到头来才发现 C 连对象都没有 :sob:

Q：可能遇到哪些语法库上的困难？

A：困难可能也算不上，但没有原生的哈希和动态数组真滴有点难受（一只菜鸟的独白 :sob:

Q：推荐用其他语言写吗？

A：当然推荐，例如 C++/Java/Python/Go...
